# .386P (32-bit MASM)

Enables assembly of all instructions (including privileged) for the 80386 processor; disables assembly of instructions introduced with later processors. (32-bit MASM only.)

## Syntax

> **.386P**

## Remarks

Also enables 80387 instructions.
