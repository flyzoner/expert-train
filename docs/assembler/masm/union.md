# UNION

Declares a union of one or more data types. The *field-declarations* must be valid data definitions. Omit the [ENDS](ends-masm.md) *name* label on nested **UNION** definitions.

## Syntax

> *name* **UNION** ⟦*alignment*⟧ ⟦__,__ **NONUNIQUE**⟧\
> *field-declarations*\
> ⟦*name*⟧ **ENDS**
