# ELSEIF

Combines [ELSE](else-masm.md) and [IF](if-masm.md) into one statement.

## Syntax

> **ELSEIF** *constantExpression*\
> *statements*\
> ⟦**ELSE**\
> *else-statements*⟧\
> **ENDIF**

## Remarks

See [IF](if-masm.md) for more information.
