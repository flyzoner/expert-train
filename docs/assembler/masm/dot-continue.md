# .CONTINUE (32-bit MASM)

Generates code to jump to the top of a [.WHILE](dot-while.md) or [.REPEAT](dot-repeat.md) block if *condition* is true. (32-bit MASM only.)

## Syntax

> **.CONTINUE** ⟦**.IF** *condition*⟧
