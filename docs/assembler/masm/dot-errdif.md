# .ERRDIF, .ERRDIFI

Generates an error if the text items are different.

## Syntax

> **.ERRDIF** *textitem1*__,__ *textitem2* ⟦__,__ *message*⟧\
> **.ERRDIFI** *textitem1*__,__ *textitem2* ⟦__,__ *message*⟧

## Remarks

If **.ERRDIFI** is given, the comparison is case insensitive.
