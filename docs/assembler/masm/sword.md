# SWORD

Allocates and optionally initializes a signed word (2 bytes) of storage for each *initializer*. Can also be used as a type specifier anywhere a type is legal.

## Syntax

> *name* **SWORD** *initializer* ⟦__,__ *initializer* ...⟧
