# LABEL

Creates a new label by assigning the current location-counter value and the given *qualifiedType* to *name*.

## Syntax

> *name* **LABEL** *qualifiedType*
>
> *name* **LABEL** ⟦**NEAR** | **FAR** | **PROC**⟧ **PTR** ⟦*qualifiedType*⟧
