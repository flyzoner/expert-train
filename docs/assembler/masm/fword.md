# FWORD

Allocates and optionally initializes 6 bytes of storage for each *initializer*.

## Syntax

> ⟦*name*⟧ **FWORD** *initializer* ⟦__,__ *initializer* ...⟧

## Remarks

Also can be used as a type specifier anywhere a type is legal.
