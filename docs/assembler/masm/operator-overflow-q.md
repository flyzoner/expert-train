# operator OVERFLOW?

Status of overflow flag. Used only within [.IF](dot-if.md), [.WHILE](dot-while.md), or [.REPEAT](dot-repeat.md) blocks and evaluated at run time, not at assembly time.

## Syntax

> **OVERFLOW?**
