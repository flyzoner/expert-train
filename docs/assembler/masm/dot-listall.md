# .LISTALL

Starts listing of all statements.

## Syntax

> **.LISTALL**

## Remarks

Equivalent to the combination of [.LIST](dot-list.md), [.LISTIF](dot-listif.md), and [.LISTMACROALL](dot-listmacroall.md).
