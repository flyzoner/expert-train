# .BREAK (32-bit MASM)

Generates code to terminate a [.WHILE](dot-while.md) or [.REPEAT](dot-repeat.md) block if *condition* is true. (32-bit MASM only.)

## Syntax

> **.BREAK** ⟦**.IF** *condition*⟧
