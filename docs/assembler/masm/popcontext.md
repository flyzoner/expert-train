# POPCONTEXT

Restores part or all of the current *context* (saved by the [PUSHCONTEXT](pushcontext.md) directive). The *context* can be **ASSUMES** (32-bit MASSM only), **RADIX**, **LISTING**, **CPU** (32-bit MASSM only), or **ALL**.

## Syntax

> **POPCONTEXT** *context*
