# .CONST (32-bit MASM)

When used with [.MODEL](dot-model.md), starts a constant data segment (with segment name **CONST**).

## Syntax

> **.CONST**\
> ⟦ *segmentItem* ⟧...

## Remarks

This segment has the read-only attribute.
