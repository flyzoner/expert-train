# .NOCREF

Suppresses listing of symbols in the symbol table and browser file.

## Syntax

> **.NOCREF** ⟦*name*⟦__,__ *name* ...⟧⟧

## Remarks

If names are specified, then only the given names are suppressed. Same as [.XCREF](dot-xcref.md).
