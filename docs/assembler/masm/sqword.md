# SQWORD

Allocates and optionally initializes 8 signed bytes of storage for each *initializer*. Also can be used as a type specifier anywhere a type is legal.

## Syntax

> *name* **SQWORD** *initializer* ⟦__,__ *initializer* ...⟧
