# `_bstr_t::operator !`

**Microsoft Specific**

Checks if the encapsulated `BSTR` is a NULL string.

## Syntax

```cpp
bool operator!( ) const throw( );
```

## Return Value

It returns **`true`** if the encapsulated `BSTR` is a NULL string, **`false`** if not.

**END Microsoft Specific**
